# AgenticCLI

Scaffold inicial do projeto `agentic` em Swift nativo com SwiftPM, preparado para evolução por fases (Fase 0, Fase 1, Fase 2/v1.0).

## Objetivo

- Binário único: `agentic`
- Modos: CLI (`run`, `auth`, `model`, `session`, `mcp`) e interativo (`chat`)
- Núcleo desacoplado da TUI
- Providers adapter-first
- Keychain + SQLite + MCP (stdio)
